# 김진원의 깃 공부하기

## HTML이란?

    -  HTML은 HyperText Markup Language의 약자입니다.
     웹 페이지는 HTML 문서라고도 불리며, HTML 태그들로 구성됩니다.
     각각의 HTML 태그는 웹 페이지의 디자인이나 기능을 결정하는데 사용됩니다.

## HTML 기본 구성

<!DOCTYPE html>
<html>
    <head>
         <meta charset="UTF-8">
         <title> html문서의 이름</title>  
    </head>

    <body>
      <h1> 제목 1 크기이다.</h1>
      <h2> 제목 2 크기이다.</h2>
      <h3> 제목 3 크기이다.</h3>
    </body>

</html>
